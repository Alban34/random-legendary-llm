# Legendary: Marvel Randomizer — v1.0.3 Release Notes

**Released: April 18, 2026**

> Fan-made tool. Not affiliated with Marvel or Upper Deck Entertainment.

## What is this?

Legendary: Marvel Randomizer is a free, browser-based companion app for players of *Legendary: A Marvel Deck-Building Game*. Open the URL. Get a legal, fresh, collection-aware game setup in seconds. No install. No account. No tracking. Ever.

**[Try it now →](https://alban34.github.io/random-legendary-llm)**

---

## What's in v1.0.3

This release focuses on accuracy and completeness: six missing expansions are now in the catalog, solo mode now enforces two scheme eligibility rules from the rulebook, every supported language is fully translated, and two persistent bugs — a broken "Clear selection" button and an unreliable PWA install prompt — are fixed.

---

### Six new expansions added to the catalog

The game data has been audited against BoardGameGeek and the following expansions are now fully included in the randomizer:

- **Weapon X** (2024)
- **Ant-Man and the Wasp** (2024)
- **2099** (2024)
- **Midnight Sons** (2023)
- **The Infinity Saga** (2023)
- **Marvel Studios' What If...?** (2023)

If you own any of these sets, add them to your collection from the Browse tab — they will appear in your collection tracking, freshness scoring, and random draws immediately. Note that *Marvel Studios' What If...?* is a standalone base game, not an expansion. The release year for **World War Hulk** has also been corrected from 2019 to 2018.

---

### Solo mode scheme eligibility rules enforced

Two schemes are incompatible with standard solo play and are now excluded accordingly:

- **Super Hero Civil War** (Core Set and Marvel Studios, Phase 1)
- **Negative Zone Prison Breakout** (Core Set)

Neither scheme can appear in a randomly generated standard solo setup, and both are disabled in the forced-selection panel when standard solo is the active mode. Switching to any other play mode restores their availability.

---

### "Clear selection" regression fixed

In the Active Filter panel on the New Game tab, clicking **Clear selection** was incorrectly checking every expansion checkbox instead of unchecking them — the opposite of the intended behaviour. This is fixed. Clear selection now reliably unchecks every expansion checkbox. **Use all expansions** continues to restore all checkboxes to the checked state. A dedicated regression test has been added to prevent this from recurring.

---

### Full translation coverage across all six languages

Every user-facing string is now translated in all six supported languages: English, French, German, Japanese, Korean, and Spanish. Several strings introduced by recent features had been displaying in English regardless of the selected language. All gaps are closed — no string falls back to English when a non-English locale is active.

---

### PWA install prompt reliability improved

The manifest link in the app source was using a root-relative path (`/manifest.webmanifest`) instead of the Vite base-path-aware form. While Vite's build step happened to rewrite this correctly in most cases, the wiring was fragile. It has been corrected to the canonical `%BASE_URL%manifest.webmanifest` form, ensuring the PWA install prompt surfaces reliably across all browsers and build configurations.
